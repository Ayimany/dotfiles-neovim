# Neovim Dotfiles
This is a collection of files pertaining to my [neovim](https://neovim.io/) configuration

## Plugins:
- **Autopair** support: [autopairs](https://github.com/windwp/nvim-autopairs)
- **Theme**: [catppuccin](https://github.com/catppuccin/nvim)
- Code **completion** menu: [cmp](https://github.com/catppuccin/nvim)
- **Git management**: [fugitive](https://github.com/tpope/vim-fugitive)
- Bufferline **Git symbols**: [gitsigns](https://github.com/lewis6991/gitsigns.nvim)
- Extensive **hover** support: [hover](https://github.com/lewis6991/hover.nvim)
- **LSP configuration**: [lspconfig](https://github.com/neovim/nvim-lspconfig)
- **LSP** completion **icons**: [lspkind](https://github.com/onsails/lspkind.nvim)
- **LSP** server **installation**: [mason](https://github.com/williamboman/mason.nvim)
- Quick **file finding**: [telescope](https://github.com/nvim-telescope/telescope.nvim)
- **Treesitter syntax** highlighting: [treesitter](https://github.com/nvim-treesitter/nvim-treesitter)
- **LSP information** menu: [trouble](https://github.com/folke/trouble.nvim)
